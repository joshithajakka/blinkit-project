Blinkit Sales Dashboard 📊

Project Overview

This project presents an interactive sales dashboard for Blinkit – India's Last Minute App.
The dashboard provides insights into sales performance, product distribution, outlet characteristics, and customer ratings. It helps businesses understand key metrics and make data-driven decisions.

The dashboard was created to analyze Blinkit's retail sales data and visualize important trends such as outlet performance, product categories, and fat content distribution.

---

Features & Visualizations

1. Sales by Fat Content

Displays the total sales split between:

- Low Fat
- Regular

This helps identify customer preference based on product nutritional categories.

2. Item Type Analysis

Shows sales contribution by different product categories such as:

- Fruits & Vegetables
- Snack Foods
- Household Items
- Frozen Foods
- Dairy Products
- Canned Goods
- Baking Goods
- Health & Hygiene

3. Outlet Establishment Trend

A time-based chart showing the growth of outlets over the years and their impact on total sales.

4. Outlet Size Distribution

Shows how sales are distributed among:

- Small outlets
- Medium outlets
- High/Large outlets

5. Outlet Location Analysis

Sales comparison across:

- Tier 1 cities
- Tier 2 cities
- Tier 3 cities

6. Outlet Type Performance

Comparison between different outlet types such as:

- Grocery Stores
- Supermarkets

Metrics shown include:

- Total Sales
- Number of Items
- Average Sales
- Average Rating
- Item Visibility

---

Filters

The dashboard includes interactive filters for:

- Outlet Location Type
- Outlet Size
- Item Type

These allow users to dynamically explore the dataset.

---

Tools & Technologies Used

- Power BI – Dashboard development and visualization
- Microsoft Excel / Dataset – Data source
- Data Cleaning & Transformation – Power Query

---

Business Insights

Some insights derived from the dashboard:

- Supermarkets generate significantly higher sales compared to grocery stores.
- Tier 3 locations contribute the highest total sales.
- Low fat products contribute a large portion of overall sales.
- Fruits, snacks, and household items are among the top-selling categories.

---

Project Purpose

The goal of this project is to demonstrate:

- Data analysis skills
- Data visualization techniques
- Business insight generation
- Dashboard design using Power BI

This project can be useful for data analyst portfolios and learning retail sales analytics.
